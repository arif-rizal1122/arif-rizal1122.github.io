6. ⚠️ Catatan Penting
Aspek

Catatan

Keamanan

Jangan simpan data sensitif di kode frontend secara hardcoded.

LocalStorage

Data dashboard pribadi hanya tersimpan di browser pengguna (tidak sinkron antar perangkat).

Formulir

Gunakan Formspree/EmailJS karena GitHub Pages tidak support backend.

SEO

Pastikan setiap halaman punya <title> dan <meta description> yang unik.

Base URL

Jika deploy di username.github.io/portfolio, set base di config framework ke /portfolio.
