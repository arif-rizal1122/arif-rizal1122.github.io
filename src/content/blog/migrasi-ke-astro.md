# Migrasi Portofolio ke Astro

Dalam postingan ini, saya akan berbagi pengalaman saya memindahkan situs portofolio statis saya ke **Astro**.

## Mengapa Astro?

Astro sangat ringan karena hanya mengirimkan JavaScript seminimal mungkin ke browser (Islands Architecture). Ini sangat cocok untuk situs seperti portofolio.

## Fitur Baru

Dengan migrasi ini, saya berhasil menambahkan:
- Dark/Light Mode
- Filter Proyek Dinamis
- Dashboard Pribadi

Tunggu perkembangan selanjutnya!
